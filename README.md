CalcPicVault is a secure media storage app cleverly disguised as a calculator. Protect your sensitive images and videos with ease and privacy.

Features
Secure Media Storage: Encrypts and stores your photos and videos.
Disguised Interface: Access your vault by entering a custom PIN on the calculator.
Organization: Create and manage albums for your media.
Privacy-Focused: All data is stored locally and encrypted.
Permissions
CalcPicVault requests the Manage All Files permission to:

Access media files for encryption and secure storage.
Ensure smooth import/export functionality.
Your data remains private and secure, with no external access or cloud storage.

How to Build
Download all files.
Open the project in Android Studio.
Build and run the app on your Android device.
